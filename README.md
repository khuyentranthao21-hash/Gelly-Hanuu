# Gelly Hanuu

A lightweight English-language HTML5 arcade game.

## Run locally
Open `index.html` in a modern browser.

## Controls
- Mouse or touch: move the catcher
- Left/Right arrows or A/D: move the catcher
- Space: pause/resume

## Purchases
The game contains two consumable purchase entries:
- $5 Jelly Boost
- $10 Jelly Mega Boost

Browser mode simulates purchases for testing. Real payments require Microsoft Store add-on integration.
